# Vue Express Map JSON Data

## Run The backend
```
cd backend
npm i
node app.js
```

## Run The frontend for development
```
cd frontend
nvm use 15.3.0
npm i
npm run serve
// the front-end needs to be updated in the future due to insecure packages
```

### Compiles and minifies for production
```
npm run build
```
